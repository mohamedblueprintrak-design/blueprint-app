# 🪄 BluePrint Engineering OS

**نظام تشغيل هندسي ذكي يعمل بالذكاء الاصطناعي**  
الإصدار: 9.0 (مارس 2026)

---

## 📋 نبذة عن المشروع

BluePrint Engineering OS هو منصة متكاملة لإدارة المشاريع الهندسية، تجمع بين التصميم الإنشائي، حصر الكميات، إدارة العيوب، تقارير الموقع، والذكاء الاصطناعي التوليدي. صُمم لمساعدة المهندسين والمقاولين ومديري المشاريع في مهامهم اليومية، مع التركيز على توفير الوقت وزيادة الكفاءة.

---

## ✨ الميزات الرئيسية

### 🏗️ إدارة المشاريع والمستخدمين
- نظام صلاحيات متكامل (Admin، Engineer، Viewer).
- إنشاء مشاريع جديدة مع تحديد الموقع.
- ربط المستخدمين بالمشاريع بصلاحيات مختلفة.
- إعدادات خاصة لكل مشروع (أسعار المواد، النموذج المفضل للذكاء الاصطناعي).

### 💬 الذكاء الاصطناعي والمحادثة (Blue)
- دعم متعدد النماذج: Gemini، GPT-4o، DeepSeek، Grok، OpenRouter، Mistral AI.
- وضع محاكاة (Mock) تلقائي عند فشل جميع المفاتيح – يضمن استمرارية العمل.
- **الذكاء السياقي**: Blue يفهم سياق المشروع الحالي (مساحته، عيوبه، تكلفته) ليقدم إجابات أكثر دقة.
- سجل محادثة يحفظ السياق.

### 📐 التصميم الإنشائي
- تحليل الكمرات مع ثلاثة سيناريوهات (اقتصادي، خفيف، آمن).
- رسم مقطع الكمرة مع توزيع حديد التسليح.
- تصدير رسم DXF للكمرات.
- تحليل البلاطات، الأعمدة، الأساسات، الجدران الاستنادية، السلالم.
- محرك قواعد للسلامة الإنشائية (تحذيرات للتصاميم غير الآمنة).

### 📊 حصر الكميات (BOQ)
- عرض جدول الكميات مع إمكانية الإضافة والحذف.
- تصدير إلى Excel.
- حساب التكلفة التقديرية.

### 🔍 إدارة العيوب
- تسجيل العيوب (وصف، شدة، حالة).
- تعديل وحذف العيوب.
- فلترة حسب الحالة والشدة، بحث في الوصف.
- **زر مشاركة عبر واتساب** بجانب كل عيب للتواصل الفوري مع الفريق.
- إحصائيات سريعة وإصدار تقارير نصية.

### 📍 تقارير الموقع
- تسجيل زيارات الموقع مع صور وإحداثيات GPS.
- **زر "📍 استخدم موقعي الحالي"** للحصول على الموقع تلقائياً (مثل واتساب).
- استخراج بيانات EXIF من الصور (التاريخ، الإحداثيات).
- عرض التقارير السابقة مع الصور المصغرة.

### 🏥 مؤشر صحة المشروع (Project Health Score)
- مؤشر بصري (أخضر/أصفر/أحمر) يظهر في لوحة المعلومات.
- يعكس صحة المشروع بناءً على العيوب المفتوحة، التأخير، وتجاوز الميزانية.
- يساعد المديرين على اتخاذ قرارات سريعة.

### 📅 تقارير يومية تلقائية (Daily Auto Report)
- نظام يرسل كل مساء (الساعة 6:00) تقرير PDF إلى مدير المشروع (بريد إلكتروني أو واتساب).
- يحتوي التقرير على: ملخص اليوم، العيوب الجديدة، الكميات المضافة، تقدم المشروع، و Health Score.
- **هذه الميزة تجعل التطبيق عادة يومية لا غنى عنها.**

### 📱 دعم الموبايل (PWA)
- يمكن تثبيت التطبيق على شاشة الهاتف الرئيسية كتطبيق مستقل.
- يعمل بشكل أسرع ويدعم العمل الجزئي بدون إنترنت.

### 📚 قاعدة معرفية هندسية
- معادلات إنشائية أساسية.
- كودات البناء الشائعة (ECP، ACI، BS، Eurocode).
- نسب خلط الخرسانة التقريبية.
- أسئلة شائعة.

### 📄 تقارير PDF
- تصدير تقرير شامل للمشروع (معلومات، تحليلات، عيوب، كميات).
- دعم الكتابة باللغة العربية.

### ⚙️ تحسينات الأداء والأمان
- تخزين مؤقت (Caching) باستخدام Redis.
- دعم Azure Key Vault للمفاتيح الحساسة.
- نقاط نهاية API موثقة.

## 🛠️ التقنيات المستخدمة

- **الواجهة الأمامية:** Streamlit (Python)
- **الخادم الخلفي:** FastAPI (Python)
- **قاعدة البيانات:** SQLite + SQLAlchemy ORM
- **الذكاء الاصطناعي:** Gemini، OpenAI، DeepSeek، Grok، OpenRouter، Mistral
- **الرسومات:** Matplotlib، Plotly
- **CAD:** ezdxf
- **معالجة الصور:** exifread
- **التقارير:** ReportLab (مع دعم العربية)
- **الأمان:** JWT، passlib، python-jose
- **التخزين المؤقت:** Redis (اختياري)
- **إدارة المفاتيح:** Azure Key Vault (اختياري)
- **الجدولة:** APScheduler

- (اختياري) حساب Azure Key Vault

## 🔧 التثبيت والإعداد

### 1. استنساخ المستودع (أو فك ضغط الملفات)
```bash
git clone <url>
cd engineering-os-ai
2. إنشاء بيئة افتراضية (اختياري)
bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
3. تثبيت المتطلبات
bash
pip install -r requirements.txt
4. إعداد ملف .env
أنشئ ملف .env في المجلد الرئيسي وأضف المتغيرات التالية (يمكنك نسخ .env.example):

env
# أساسي
SECRET_KEY=your-secret-key-change-this-in-production

# مفاتيح API (اختياري - إذا لم توفرها، سيعمل وضع المحاكاة)
OPENAI_API_KEY=sk-...
GEMINI_API_KEY=AI...
DEEPSEEK_API_KEY=sk-...
GROK_API_KEY=xai-...
OPENROUTER_API_KEY=sk-or-...
MISTRAL_API_KEY=...

# Redis (اختياري)
REDIS_URL=redis://localhost:6379

# Azure Key Vault (اختياري)
AZURE_VAULT_URL=https://your-vault.vault.azure.net/
AZURE_TENANT_ID=...
AZURE_CLIENT_ID=...
AZURE_CLIENT_SECRET=...
5. إعداد قاعدة البيانات
bash
# لا حاجة لتشغيل migrations، التطبيق ينشئ الجداول تلقائياً عند أول تشغيل.
🚀 تشغيل التطبيق
الطريقة اليدوية (نافذتان منفصلتان)
النافذة 1 (الخادم الخلفي):

bash
uvicorn main:app --reload
النافذة 2 (الواجهة الأمامية):

bash
streamlit run gateway.py
الطريقة التلقائية (ملف دفعي)
أنشئ ملف run_all.bat في المجلد الرئيسي بالمحتوى التالي:

batch
@echo off
title BluePrint Engineering OS - Launcher
echo ========================================
echo    تشغيل BluePrint Engineering OS
echo ========================================
echo.
echo [1] بدء تشغيل الخادم الخلفي (FastAPI)...
start "BluePrint Server" cmd /k "cd /d %~dp0 && uvicorn main:app --reload"
timeout /t 3 /nobreak >nul

echo [2] بدء تشغيل الواجهة الأمامية (Streamlit)...
start "BluePrint UI" cmd /k "cd /d %~dp0 && streamlit run gateway.py"

echo.
echo ✅ تم بدء تشغيل جميع المكونات.
echo الخادم: http://127.0.0.1:8000
echo الواجهة: http://localhost:8501
echo.
echo يرجى عدم إغلاق هذه النافذة.
pause
ثم انقر نقراً مزدوجاً على run_all.bat لتشغيل كل شيء بنقرة واحدة.
